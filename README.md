# AI Image Engineering

This project documents my journey building a local AI image editing pipeline using:

- ComfyUI
- SDXL (low VRAM)
- ControlNet
- IP-Adapter

Hardware: NVIDIA GTX 1650 (4GB VRAM)

## Goals
- Run SDXL locally
- Build Grok-style image editing workflows
- Use ControlNet for structure control
- Use IP-Adapter for style and identity transfer
- Document everything clearly
